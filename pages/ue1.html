UE1. Réaliser un développement d'application

Développement d'Application avec Gestion de Données

Dans le cadre de mon Unité d'Enseignement 1, j'ai eu l'opportunité de m'engager dans un projet passionnant de développement d'application. Ce projet, qui s'inscrivait dans la continuité d'un travail pratique antérieur, visait à réaliser une application web capable de gérer (ajouter, modifier, supprimer, afficher) les données contenues dans deux tableaux Python spécifiques. Le but était de parfaire nos compétences pratiques en développement web tout en respectant un cahier des charges précis.

Objectifs du Projet

L'objectif principal était de concevoir et développer des interfaces utilisateur intuitives pour la manipulation des données des tableaux Python, en assurant une expérience utilisateur fluide et ergonomique. En outre, le projet insistait sur l'importance du respect des bonnes pratiques de développement, notamment :
- Utilisation du snake_case pour les noms des fonctions et du kebab-case pour les routes.
- Organisation logique des fichiers de template.
- Validation des contenus des pages web selon les standards W3C.

Compétences Développées

Conception d'Interface Utilisateur : J'ai conçu des interfaces utilisateur ergonomiques, mettant en avant ma capacité à anticiper les besoins des utilisateurs pour une expérience fluide et intuitive.

Programmation Web : J'ai approfondi mes compétences en HTML, CSS, et JavaScript, créant des pages web dynamiques qui interagissent efficacement avec le serveur Python en backend pour la gestion des données.

Respect des Normes et Bonnes Pratiques : J'ai rigoureusement suivi les consignes de nomenclature, structuré le projet de manière logique, et veillé à ce que toutes les pages soient valides selon les critères du W3C, soulignant mon engagement envers la qualité du code.

Gestion de Données : Le cœur du projet résidait dans la manipulation des données via les opérations CRUD, sans altérer les tableaux Python originaux, démontrant ma capacité à gérer efficacement les données dans une application web.

Ajouter un hôtel : On peut rapidement ajouter un nouvel hôtel en remplissant un formulaire simple. C'est parfait pour élargir la liste avec de nouveaux établissements.

Éditer les informations : Si vous devez mettre à jour les détails d'un hôtel, comme son adresse ou son classement, l'option d'édition rend cela facile. Quelques clics et les informations sont à jour.

Supprimer un hôtel : Pour retirer un hôtel de la liste, il suffit d'utiliser le bouton "Supprimer". Une confirmation est demandée pour s'assurer qu'aucune suppression n'est faite par erreur.

Conclusion

Ce projet m’a permis de consolider mes compétences en développement web tout en me familiarisant avec les principes de conception d'interfaces utilisateur et le respect des normes de développement. Il a renforcé ma compréhension des défis liés à la création d'applications web modernes et accessibles, me préparant ainsi pour des projets futurs plus ambitieux.
